# Goethe-Zertifikat B1 Module Documentation
## Overview

GoetheForge B1 provides comprehensive preparation for the Goethe-Zertifikat B1 exam.
## Lesen (Reading) Module

- Duration: 65 minutes
- Parts: 5 parts (30 items)
- Pass threshold: 60/100 points
## H—ren (Listening) Module

- Duration: 40 minutes
- Parts: 4 parts (30 items)
- Play limits: Teil 1 & 4 (2x), Teil 2 & 3 (1x)
